Exploring the Document Object Model (DOM)

The Document Object Model (DOM) is a programming interface for web documents. It represents the structure of an HTML or XML document as a tree of objects that can be manipulated with JavaScript.

DOM Manipulation

DOM manipulation refers to the process of accessing or modifying elements within an HTML document using JavaScript.

// Example of accessing an element by ID
var element = document.getElementById('myElementId');
element.textContent = 'Updated text content';

DOM Events

DOM events are actions or occurrences that happen in the DOM. They can trigger JavaScript functions or scripts to respond to user actions or other events.

// Example of adding an event listener
var button = document.getElementById('myButton');
button.addEventListener('click', function() {
    alert('Button clicked!');
});
