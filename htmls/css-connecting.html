Introduction to CSS

CSS (Cascading Style Sheets) is used to style and layout web pages. It can be connected to HTML in several ways.

Inline CSS

Inline CSS is used to apply a unique style to a single HTML element. It uses the style attribute:

<p style="color:blue;">This is a blue paragraph.</p>

Internal CSS

Internal CSS is used to define styles within the <style> element, inside the <head> section of an HTML document:

<head>
    <style>
        p {
            color: red;
        }
    </style>
</head>
<body>
    <p>This is a red paragraph.</p>
</body>

External CSS

External CSS is used to define styles in a separate CSS file. The CSS file is then linked to the HTML document using the <link> element:

<head>
    <link rel="stylesheet" href="styles.css">
</head>
<body>
    <p>This is a styled paragraph.</p>
</body>

The external CSS file (styles.css) might contain:

p {
    color: green;
}

Multiple Stylesheets

You can link multiple external stylesheets to a single HTML document:

<head>
    <link rel="stylesheet" href="styles1.css">
    <link rel="stylesheet" href="styles2.css">
</head>
