Introduction to CSS Colors

CSS colors can be defined using various formats, including HEX and RGB values. These formats are used to set the color of elements in a webpage.

HEX Colors

HEX color values are specified with a hash (#) followed by six hex digits, representing the intensity of red, green, and blue. Example:

/* HEX color example */
p {
    color: #ff0000; /* Red */
    background-color: #00ff00; /* Green */
}

This text is red with a green background using HEX colors.

RGB Colors

RGB color values are specified with the rgb() function, which defines the intensity of red, green, and blue. Example:

/* RGB color example */
p {
    color: rgb(255, 0, 0); /* Red */
    background-color: rgb(0, 255, 0); /* Green */
}

This text is red with a green background using RGB colors.
