Introduction to CSS Border

The border property is used to set the border around an element. You can specify the width, style, and color of the border.

Border Example

Here is an example of setting a border in CSS:

/* Border example */
p {
    border: 2px solid black;
}

This paragraph has a 2px solid black border.

Different Border Styles

You can use different border styles like dashed, dotted, double, etc.:

/* Different border styles */
p {
    border: 2px dashed blue;
}

This paragraph has a 2px dashed blue border.

Individual Border Properties

You can set borders individually for each side:

/* Individual border properties */
p {
    border-top: 2px solid red;
    border-right: 4px dotted green;
    border-bottom: 6px double blue;
    border-left: 8px groove purple;
}

This paragraph has different borders on each side (top: 2px solid red, right: 4px dotted green, bottom: 6px double blue, left: 8px groove purple).
