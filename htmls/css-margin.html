Introduction to CSS Margin

Margin is the space outside the border of an element. It can be set for each side of the element (top, right, bottom, left).

Margin Example

Here is an example of setting margin in CSS:

/* Margin example */
p {
    margin: 20px; /* Applies 20px margin on all sides */
    border: 1px solid black;
}

This paragraph has 20px margin on all sides.

Individual Margin Properties

Margin can be set individually for each side:

/* Individual margin properties */
p {
    margin-top: 10px;
    margin-right: 20px;
    margin-bottom: 30px;
    margin-left: 40px;
    border: 1px solid black;
}

This paragraph has different margin on each side (top: 10px, right: 20px, bottom: 30px, left: 40px).
