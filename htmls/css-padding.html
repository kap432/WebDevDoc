Introduction to CSS Padding

Padding is the space between the content of an element and its border. It can be set for each side of the element (top, right, bottom, left).

Padding Example

Here is an example of setting padding in CSS:

/* Padding example */
p {
    padding: 20px; /* Applies 20px padding on all sides */
    border: 1px solid black;
}

This paragraph has 20px padding on all sides.

Individual Padding Properties

Padding can be set individually for each side:

/* Individual padding properties */
p {
    padding-top: 10px;
    padding-right: 20px;
    padding-bottom: 30px;
    padding-left: 40px;
    border: 1px solid black;
}

This paragraph has different padding on each side (top: 10px, right: 20px, bottom: 30px, left: 40px).
