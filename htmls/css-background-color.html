CSS Background Color

The background-color property in CSS is used to specify the background color of an element. You can use color names, HEX values, RGB values, or HSL values to set the background color.

Background Color Example

Here is an example of setting a background color using a color name:

/* Background color using a color name */
p {
    background-color: lightblue;
}

This paragraph has a light blue background color.

HEX Color

You can set the background color using HEX values:

/* Background color with HEX */
p {
    background-color: #ff6347; /* Tomato */
}

This paragraph has a tomato background color (HEX: #ff6347).

RGB Color

You can also set the background color using RGB values:

/* Background color with RGB */
p {
    background-color: rgb(255, 99, 71); /* Tomato */
}

This paragraph has a tomato background color (RGB: 255, 99, 71).
