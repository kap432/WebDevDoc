Introduction to JavaScript

JavaScript is a scripting language used to create and control dynamic website content, interact with users, and much more.

console.log()

The console.log() method is used to log or print messages to the console, typically for debugging purposes.

/* Example of console.log() */
console.log('Hello, world!');

Variables: let, var, and const

Variables in JavaScript are containers that store data values. There are three ways to declare variables: let, var, and const.

let

let allows you to declare variables that are limited to the scope of a block statement or expression in which it is used. It is a good practice to use let over var for variable declaration in modern JavaScript.

/* Example of let */
let message = 'Hello, world!';
console.log(message);

var

var is the older way to declare variables in JavaScript. It is globally scoped or function scoped which can lead to unintended behavior when used inside blocks.

/* Example of var */
var count = 10;
console.log(count);

const

const is used to declare variables that cannot be reassigned. The value of a constant cannot change through reassignment, and it must be initialized during declaration.

/* Example of const */
const pi = 3.14;
console.log(pi);

Functions

Functions are reusable blocks of code that perform a specific task. They can be defined using the function keyword.

Defining a Function

To define a function, use the function keyword followed by the function name and parentheses () containing optional parameters.

/* Example of defining a function */
function greet(name) {
    return 'Hello, ' + name + '!';
}

Calling a Function

To execute or call a function, use the function name followed by parentheses ().

/* Example of calling a function */
console.log(greet('Alice')); // Outputs: Hello, Alice!
