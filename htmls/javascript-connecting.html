Connecting JavaScript to HTML Pages

JavaScript can be connected to HTML pages using different techniques to enhance interactivity and functionality.

Inline Scripting

Inline scripting involves embedding JavaScript directly within HTML using the <script> tag's onclick, onload, or other event attributes.

<button onclick="alert('Button clicked!')">Click Me</button>

Internal Scripting

Internal scripting places JavaScript code within the <script> tags directly in the HTML document's <head> or <body> section.

<script>
    function greet() {
        alert('Hello, world!');
    }
</script>

External Scripting

External scripting links an external JavaScript file using the <script src="script.js"></script> tag, enabling reuse and separation of concerns.

<script src="script.js"></script>
